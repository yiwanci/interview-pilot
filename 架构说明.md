# Interview Pilot 项目架构说明

## 项目概述

Interview Pilot 是一个基于 **Agent + RAG + 记忆系统** 的 AI 面试学习助手。

### 核心能力
- 🧠 **AI学习助手**：RAG 检索知识库 + 对话历史，智能讲解技术知识点
- 📚 **智能记忆系统**：SM-2 遗忘曲线算法，自动生成复习计划
- 🎯 **AI模拟面试**：根据薄弱知识点和遗忘曲线生成个性化面试
- 🕷️ **自动采集面经**：从小红书、抖音等平台爬取面试经验

## 核心架构流程

### 1. Agent 工作流（LangGraph）

```
用户输入
    ↓
router_node（意图识别）
    ↓
    ├─→ study_node（学习讲解）
    ├─→ interview_node（模拟面试）
    ├─→→ crawler_node（爬取面经）
    ├─→ plan_node（学习计划）
    └─→ chat_node（闲聊）
    ↓
返回 response
```

**关键文件：`agent/graph.py`**

### 2. 状态传递（AgentState）

所有节点通过 `AgentState` 传递数据：

```python
class AgentState(TypedDict):
    user_input: str                    # 用户输入
    intent: str                        # 意图识别结果
    conversation_history: List[dict]  # 对话历史
    session_id: str                    # 会话标识
    memory_context: dict              # 记忆上下文
    rag_results: list                 # RAG 检索结果
    response: str                     # 最终输出
    # ... 更多字段见 state.py
```

**关键文件：`agent/state.py`**

### 3. 记忆系统架构

```
MemoryManager（统一接口）
    ├─→ SQLiteStore（结构化数据）
    │   ├─ 知识点（KnowledgePoint + SM-2）
    │   ├─ 学习日志（StudyLog）
    │   └─ 用户画像（UserProfile）
    │
    ├─→ Mem0Client（语义记忆）
    │   └─ Qdrant 向量存储
    │
    └─→ SM2Engine（遗忘曲线算法）
```

**关键文件：`memory/memory_manager.py`**

       # 1. 处理逻辑（调用 LLM、工具等）

   }
